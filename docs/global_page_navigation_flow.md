# SunnyPlay 日光陪玩 - 页面访问逻辑流程

## 1. 主要导航结构

### 底部导航栏（所有登录状态可见）
- **首页** (/): 应用默认入口
- **发现** (/discover): 寻找陪玩入口
- **社区** (/community): 社区动态入口
- **聊天** (/messages): 消息中心入口
- **订单** (/orders): 订单管理入口
- **我的** (/profile): 个人中心入口

## 2. 页面访问逻辑流程

### 2.1 基础页面

#### 首页 (Home.vue)
- **入口**：底部导航栏
- **可访问页面**：
  - 点击轮播图或推荐陪玩 → 大神详情页
  - 点击游戏分类 → 游戏分类页
  - 点击搜索框 → 搜索结果页
  - 点击顶部通知图标 → 通知页

#### 发现页 (Discover.vue)
- **入口**：底部导航栏
- **可访问页面**：
  - 点击搜索框 → 搜索结果页
  - 点击筛选按钮 → 筛选页
  - 点击陪玩卡片 → 大神详情页

#### 社区页 (Community.vue)
- **入口**：底部导航栏
- **可访问页面**：
  - 点击帖子 → 帖子详情页
  - 点击用户头像 → 大神详情页

#### 聊天页 (Messages.vue)
- **入口**：底部导航栏
- **可访问页面**：
  - 点击聊天列表项 → 聊天详情页

#### 订单页 (Orders.vue)
- **入口**：底部导航栏
- **可访问页面**：
  - 点击订单项 → 订单详情页
  - 点击待评价订单 → 评价服务页

#### 个人中心页 (Profile.vue)
- **入口**：底部导航栏
- **可访问页面**：
  - 点击编辑资料 → 编辑资料页
  - 点击设置 → 设置页
  - 点击我的钱包 → 钱包页
  - 点击我的收藏 → 收藏页
  - 点击我的关注 → 关注页
  - 点击浏览历史 → 浏览历史页
  - 点击我的技能 → 技能页
  - 点击账号安全 → 安全页
  - 点击帮助与客服 → 帮助页
  - 点击用户协议 → 用户协议页
  - 点击申诉记录 → 申诉页
  - 点击收益提现 → 提现页
  - 点击大神认证 → 大神认证页

### 2.2 认证相关页面

#### 登录页 (Login.vue)
- **入口**：
  - 未登录状态下访问需要认证的页面时自动跳转
  - 注册页、忘记密码页的返回链接
- **可访问页面**：
  - 点击注册链接 → 注册页
  - 点击忘记密码链接 → 忘记密码页

#### 注册页 (Register.vue)
- **入口**：登录页的注册链接
- **可访问页面**：
  - 点击登录链接 → 登录页

#### 忘记密码页 (ForgotPassword.vue)
- **入口**：登录页的忘记密码链接
- **可访问页面**：
  - 点击登录链接 → 登录页

### 2.3 大神相关页面

#### 大神详情页 (ExpertDetail.vue)
- **入口**：
  - 首页的推荐陪玩
  - 发现页的陪玩列表
  - 社区页的用户头像
  - 搜索结果页的陪玩列表
  - 收藏页的收藏列表
  - 关注页的关注列表
  - 浏览历史页的历史记录
- **可访问页面**：
  - 点击下单按钮 → 确认订单页
  - 点击评价标签 → 评价页

#### 大神认证页 (ExpertCertification.vue)
- **入口**：个人中心页的大神认证入口
- **可访问页面**：
  - 点击下一步 → 上传截图页

#### 上传截图页 (ExpertCertificationUpload.vue)
- **入口**：大神认证页的下一步
- **可访问页面**：
  - 点击下一步 → 录制语音页
  - 点击返回 → 大神认证页

#### 录制语音页 (ExpertCertificationVoice.vue)
- **入口**：上传截图页的下一步
- **可访问页面**：
  - 点击完成 → 个人中心页
  - 点击返回 → 上传截图页

#### 大神认证页 (ExpertVerification.vue)
- **入口**：个人中心页的大神认证入口（备用路径）
- **可访问页面**：
  - 点击完成 → 个人中心页

### 2.4 订单相关页面

## 5. 总结

通过以上页面访问逻辑流程的设计，确保了：

1. **所有页面都有入口**：每个页面至少有一个访问入口，确保用户可以通过正常操作路径访问到所有功能

2. **页面导航闭环**：所有页面都可以返回到上一级页面或主页面，形成完整的导航闭环

3. **权限控制**：需要认证的页面会在未登录状态下自动跳转到登录页

4. **用户体验**：通过底部导航栏和页面内链接，提供了直观、便捷的导航体验

5. **覆盖所有页面**：包含了项目中所有的Vue页面，确保没有遗漏

此访问逻辑流程设计符合现代移动应用的导航规范，既保证了功能的完整性，又提供了良好的用户体验。
